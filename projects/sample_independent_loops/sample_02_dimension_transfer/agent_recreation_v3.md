# sample_02_dimension_transfer - Agent Recreation v3

## Iteration 03 Focus

Resource agency: every item choice should be a strategy, not a pickup list.

### 1화 - 버릴 것을 고르는 능력

도현은 십이 분짜리 숲에서 가장 먼저 버릴 것을 골랐다.

무기는 버렸다.

무기가 있으면 싸우게 된다. 싸우면 시간을 잃는다.

그가 챙긴 것은 버스의 노선표와 정수 알약이었다. 노선표에는 숲 안쪽 정류장 이름들이 적혀 있었다.

"무기는 나중에. 오늘은 길부터."

귀환 후 노선표는 방 벽에 붙었다.

다음 문 후보가 하나 늘었다.

### 2화 - 지도 대신 빚

두 번째 이동에서 도현은 노선표가 지도처럼 작동하지 않는다는 걸 알았다.

정류장 이름을 고르면 그곳으로 갈 수 있었다.

하지만 한 번 고른 정류장은 현실 방에 빚을 남겼다. 방바닥에 숲의 습기가 스며들고, 벽지에 검은 곰팡이가 피었다.

도현은 더 먼 정류장을 고를수록 방이 망가진다는 걸 이해했다.

`거점 오염도 12%`

### 3화 - 돌아올 곳을 지키는 원정

도현은 이번에는 멀리 가지 않았다.

가까운 정류장에서 방수 천과 녹슨 못을 챙겼다. 대단한 보상은 아니었다.

하지만 방의 곰팡이를 막을 수 있었다.

생존은 강한 아이템을 모으는 일이 아니었다.

돌아올 곳을 망치지 않는 일이었다.

그때 노선표 맨 아래에 새 정류장이 생겼다.

`종점: 귀환 불가`

언젠가 저곳에 가야 한다면, 무엇을 남겨두고 떠나야 할까?
