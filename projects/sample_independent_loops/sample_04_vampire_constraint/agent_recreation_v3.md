# sample_04_vampire_constraint - Agent Recreation v3

## Iteration 03 Focus

Constraint escalation: each new ally should solve one problem and create another.

### 1화 - 박쥐의 장점

박쥐는 약했다.

대신 작았다.

도현은 그 장점 하나로 첫 던전을 깼다. 박쥐를 환풍구로 보내고, 안쪽 잠금장치를 긁게 했다.

전투는 없었다.

밖에서 본 사람들은 침투 작전이라고 불렀다.

박쥐는 그 말을 매우 마음에 들어 했다.

### 2화 - 장점은 성격을 키운다

칭찬받은 박쥐는 더 빨라졌다.

그리고 더 시끄러워졌다.

도현이 조용히 하라고 할수록 박쥐는 자신을 군단장이라고 소개했다.

소문은 전투력보다 빠르게 자랐다.

덕분에 두 번째 권속 후보가 찾아왔다.

문제는 그녀가 박쥐를 싫어한다는 점이었다.

### 3화 - 권속끼리 싸우면 주인이 강해 보인다

늑대귀 검사는 박쥐의 명령을 듣지 않았다.

박쥐도 검사를 인정하지 않았다.

도현은 둘을 말리려다 멈췄다. 둘의 싸움이 몬스터의 시선을 끌고 있었다.

그는 싸움을 지휘처럼 보이게 만들었다.

"계속해. 단, 서로 죽이지는 말고."

결과는 승리였다.

주변 헌터들은 도현이 갈등까지 계산했다고 믿었다.

다음 권속도 서로 싫어한다면, 그는 정말 지휘관이 되는 걸까?
