# sample_04_vampire_constraint - Agent Recreation v4

## Iteration 04 Focus

Evaluation readiness: each episode exposes constraint, delegated action, misunderstanding evidence, and next management debt.

### 1화 - 직접 때릴 수 없는 강자

비교 기능: 강한 이름과 약한 조건이 동시에 공개된다.

도현은 `혈왕의 대리인`이 되었다.

직접 공격은 금지였다.

첫 권속은 박쥐였다.

박쥐는 작고 약했지만, 환풍구를 통과할 수 있었다.

도현은 싸우지 않고 문을 잠갔다.

몬스터는 함정 안에서 쓰러졌다.

밖에서는 그것을 지휘라고 오해했다.

### 2화 - 오해가 계약서를 부른다

비교 기능: 외부 시선이 새 부채를 만든다.

소문은 도현보다 먼저 길드에 도착했다.

계약 희망자들이 찾아왔다.

그중 늑대귀 검사는 강했다.

조건은 까다로웠다.

`명령하면 반항. 선택지를 주면 협력.`

도현은 강한 권속을 얻는 대신, 말하는 방식까지 관리해야 했다.

### 3화 - 선택지로 싸우는 주인

비교 기능: 위임 전투가 성공하지만 명성이 더 커진다.

도현은 명령하지 않았다.

"왼쪽은 빠르고, 오른쪽은 안전하다."

검사는 오른쪽을 골랐다.

박쥐는 왼쪽에서 소리를 질렀다.

몬스터는 둘 사이에서 멈췄고, 도현은 함정을 닫았다.

주변 사람들은 성격까지 계산한 지휘라고 믿었다.

다음 권속의 조건까지 남들이 기대한다면, 도현은 언제까지 우연을 전략처럼 포장할 수 있을까?
