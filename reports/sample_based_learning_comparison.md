# 샘플 기반 학습용 1~3화 비교 리포트

Date: 2026-06-16

Target draft: `drafts/sample_based_learning_episodes_1_3.md`

## 0. 이전 초안 실패 원인

이전 1화의 실패 원인은 샘플의 재미 구조가 아니라 샘플의 표면 어휘만 따라간 데 있다.

- 사용한 것: 헌터, 게이트, 길드, 보상, 시스템 같은 단어.
- 놓친 것: 비각성자의 결핍, 갑작스러운 상태창, 생존 압박, 튜토리얼 진입, 능력의 즉시 실험, 커뮤니티 반응, 신체 제약형 클래스.
- 결과: 세계관이 세무/정산이라는 새 장치로 이동했고, 샘플군이 주는 즉시성·대리만족·생존 긴장이 약해졌다.

이번 학습용 초안은 직업 차별화보다 샘플군의 초반 독자 보상 구조를 우선했다.

## 1. 원본 샘플에서 추출한 초반 작동 구조

원문 문장은 인용하지 않고 기능만 정리한다.

| 원본 파일 | 초반 세계관 기능 | 초반 능력/장치 기능 | 독자 보상 |
|---|---|---|---|
| `무한회귀.txt` | 탑/플레이어 중심 사회, 비참여자의 강한 결핍 | 탑 구조, 게임식 성장, 선망 대상 | "나도 들어가고 싶다"는 결핍과 각성 기대 |
| `차원이동.txt` | 멸망/죽음 직전의 극한 상황 | 좌표 연결, 전송, 낯선 생존 공간 | 죽음 직전 탈출과 새 세계 진입 |
| `초월자갤러리.txt` | 낯선 회색 공간, 즉시 위험 | 던전/층 구조, 몬스터, 무기 획득 | 설명보다 먼저 생존전으로 몰입 |
| `흡혈귀.txt` | 현대 헌터 사회와 커뮤니티 반응 | EX급 클래스, 신체 변화, 약점 | 강한 클래스와 우스운 제약의 대비 |

## 2. 학습용 초안에 반영한 요소

| 반영 요소 | 학습용 초안의 처리 | 목적 |
|---|---|---|
| 비각성자의 결핍 | 주인공이 각성자 갤러리를 보며 밖에 머문다는 감각을 가짐 | 주인공의 욕망을 즉시 이해시킴 |
| 커뮤니티/갤러리 | 현실 갤러리에서 시작해 문밖 게시판으로 확장 | 독자 반응식 재미와 세계관 정보 공급 |
| 갑작스러운 상태창 | 재난 현장 자진 진입 직후 클래스 조건 발동 | 보상 루프를 빠르게 열기 |
| 전송/튜토리얼 | 피로 연 문 너머 회색 복도에 입장 | 차원이동·던전형 초반 몰입 결합 |
| 신체 제약형 능력 | 피를 흘려야 문을 여는 `혈표식` | 강한 능력에 비용 부여 |
| 즉시 전투 증명 | 문을 열고 닫아 몬스터를 절단 | 능력 설명을 행동으로 증명 |
| 이전 생존자/회귀자 암시 | 전임자, 익명 접속자, 생존자가 아닐 수 있는 게시판 | 3화 끝의 장기 미스터리 |

## 3. 1~3화 구조 비교

| 화 | 학습용 초안 기능 | 원본군과 맞는 점 | 아직 약한 점 |
|---|---|---|---|
| 1화 | 비각성 결핍 -> 게이트 발생 -> 숨겨진 클래스 -> 튜토리얼 입장 | `무한회귀`, `흡혈귀`의 질투/커뮤니티 출발과 `차원이동`식 전송 결합 | 주인공의 현실 관계가 아직 얕음 |
| 2화 | 능력 규칙 실험 -> 첫 처치 -> 보상 -> 게시판 개방 | `초월자갤러리`의 즉시 생존전, 샘플군의 시스템 보상 루프와 맞음 | 전투가 단일 복도라 시각적 변주가 적음 |
| 3화 | 전임자 등장 -> 댓글 힌트 -> 능력 응용 -> 게시판의 위험성 공개 | 회귀자/생존자 정보 격차, 갤러리식 외부 시선, 숨겨진 룰 확장 | 적의 목적과 장기 목표가 아직 완전히 잠기지 않음 |

## 4. 원본 대비 유사성 관리

보존한 것은 장르 문법과 기능이다.

- 현대 각성/탑/게이트 사회
- 상태창과 보상
- 커뮤니티 반응
- 특수 클래스와 약점
- 튜토리얼/던전형 생존전

피한 것은 보호받을 수 있는 표현과 고유 조합이다.

- 원본 고유명, 캐릭터명, 세부 지명 미사용
- 원본 문장, 대사, 장면 배열 미사용
- 원본의 특정 클래스명과 스킬명 미사용
- 원본의 사건 해결 순서 그대로 사용하지 않음

## 5. 재미 기준 평가

| 항목 | 점수 | 근거 |
|---|---:|---|
| 초반 결핍 명확성 | 4/5 | 비각성자와 갤러리 선망이 빠르게 드러남 |
| 세계관 진입 속도 | 4/5 | 1화 안에서 게이트와 튜토리얼까지 진입 |
| 능력의 직관성 | 4/5 | 피로 문을 연다는 규칙은 단순함 |
| 능력의 비용 | 4/5 | 피와 체력이 비용이라 긴장이 생김 |
| 1화 후킹 | 4/5 | 클래스 발동과 튜토리얼 입장으로 끝남 |
| 2화 보상 루프 | 4/5 | 첫 처치, 회복, 유지 시간 증가, 게시판 개방 |
| 3화 확장성 | 3/5 | 전임자와 접속자 미스터리는 생기지만 대형 목표는 더 선명해야 함 |
| 독창성 | 3/5 | 샘플 학습 목적상 의도적으로 익숙한 구조를 유지함 |

## 6. 다음 개선 방향

1. 1화 첫 페이지에 주인공의 현실 손실을 하나 더 넣는다. 예: 가족, 빚, 시험 탈락, 구조 실패 중 하나.
2. 2화 전투 공간을 복도 하나가 아니라 문이 여러 개인 방으로 바꿔 능력 응용 폭을 넓힌다.
3. 3화 끝에 장기 목표를 명시한다. 예: "일곱 접속자 중 하나가 현실 게이트 사고의 원인"처럼 추적 목표를 만든다.
4. `문밖 게시판`의 말투를 캐릭터별로 분리한다. 익명_3, 전임자, 조롱꾼, 정보상 같은 역할을 나눈다.
5. 회차당 보상을 숫자 상승보다 문제 해결 권한 상승으로 설계한다. 예: 문 열기 -> 문 닫기 -> 남의 문 열기 -> 현실 게이트 문 간섭.

## 7. 결론

이번 학습용 1~3화는 이전 세무사 초안보다 샘플군의 핵심 작동 방식에 가깝다. 특히 결핍, 상태창, 튜토리얼, 즉시 전투, 보상, 커뮤니티 미스터리를 초반 3화 안에 배치했다.

다만 아직 "장기 목표"와 "현실 손실"이 약하다. 다음 패치는 1화의 현실 동기와 3화의 추적 목표를 강화하는 쪽이 맞다.
