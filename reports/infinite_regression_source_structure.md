## 추출 기준

원문 문장, 고유명, 농담, 대사 리듬을 그대로 옮기지 않고 초반부의 기능 구조만 추출했다. 이번 재작성의 목표는 원본의 재미 작동 방식을 학습하되, 새 이름과 새 문장으로 별도 학습용 원고를 만드는 것이다.

## 초반 기능 구조

1. 세계 뉴스/방송으로 탑과 최상위 플레이어의 존재를 먼저 보여준다.
2. 주인공은 탑에 들어가지 못하는 관전자다. 그래서 현실 직업보다 탑을 향한 결핍과 집착이 먼저 드러난다.
3. 탑은 게임처럼 보이지만 실제 인류 생존과 연결된다. 클리어 포인트, 층, 보상, 아이템, 마석, 난이도 같은 규칙이 독자에게 빠르게 각인된다.
4. 주인공은 영웅의 부와 명성보다 “저 게임을 직접 하고 싶다”는 욕망을 품는다. 이 욕망이 회귀 이후의 행동 동기가 된다.
5. 인류가 필요한 포인트를 채우지 못하고 세계 멸망 절차가 시작된다.
6. 주인공은 죽음 직전에 끝을 체감하고, 탑 공식 시작일로 돌아온다.
7. 상태창은 주인공의 능력을 확인시킨다. 동시에 능력을 공개하지 말라는 금지 조건을 부여한다.
8. 회귀자 전용 커뮤니티가 등장한다. 정보, 조롱, 환영식, 경험자 조언이 섞이며 분위기를 가볍게 만든다.
9. 첫 입장 보상과 튜토리얼 기대가 제시된다.
10. 주인공의 특수성 때문에 튜토리얼 난이도가 비정상적으로 재산정되고, 초반 훅은 “첫판부터 망했다”로 닫힌다.

## 재작성 반영 규칙

- 원본의 기능 순서는 보존한다.
- 고유명은 모두 교체한다.
- 주인공의 능력은 직관적인 이름으로 둔다.
- 능력은 특정 문이나 단일 기술로 닫지 않고, 회차 축적, 실패 원인 기록, 보상 보존, 조건부 체크포인트 등으로 확장 가능한 권한형 능력으로 설계한다.
- 회귀자 커뮤니티는 정보 전달 장치이면서 코미디 장치로 사용한다.
- 튜토리얼 후반은 난이도 오류로 끝내 다음 화 클릭 이유를 만든다.

## 새 원고 설계

| 항목 | 원본 기능 | 새 원고 적용 |
|---|---|---|
| 탑 | 세계에 출현한 생존 게임 장치 | `흑첨탑` |
| 플레이어 | 선택받아 탑에 들어가는 사람 | `등반자` |
| 주인공 결핍 | 들어가지 못하는 관전자 | `서도하`, 입장 부적합 판정자 |
| 욕망 | 최종 콘텐츠를 직접 플레이하고 싶음 | 멸망보다 먼저 “한 판만”을 떠올리는 집착 |
| 능력 | 회귀 계열 핵심 권한 | `무한 리트라이` |
| 금지 조건 | 능력 공개 제한 | 리트라이 사실 발설 금지 |
| 커뮤니티 | 회귀자 모임/게시판 | `리트라이 라운지` |
| 첫 보상 | 첫 입장 혜택/랜덤 보상 | `초회 보정 상자`, `미감정 코어` |
| 훅 | 튜토리얼 난이도 비정상 상승 | 시간축 오류로 `심연 난이도` 개방 |
