## 설계 반영 원칙

- 플랫폼 규칙은 날짜와 신뢰도를 갖는 외부 프로필로 분리한다.
- 운영상 권장 수치는 공식 규정과 구분한다.
- 계약·저작권은 자동 승인 게이트가 아니라 중단·에스컬레이션 게이트로 구현한다.

## 출처 취급 주의

- 1인칭 선호, 5~8화 복선, 25~30화 이상의 비축분 등은 사용자 제공 실무 자료의 관찰·권장치이며 플랫폼 공식 규정으로 취급하지 않는다.
- 시스템은 해당 수치를 기본 휴리스틱과 경고 기준으로 사용하되, 작가의 가용시간·장르·계약 조건에 따라 조정한다.
