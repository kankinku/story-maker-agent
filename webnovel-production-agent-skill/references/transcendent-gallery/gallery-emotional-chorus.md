# Gallery Emotional Chorus

Use this reference when the gallery shifts from spectator comedy to emotional support.

## Role

The gallery starts as an audience watching a rare newbie, but gradually becomes a community that worries about the protagonist's survival and growth.

It can function as:

- mentor network
- spectator seat
- combat analysis room
- exposition channel
- comic chorus
- emotional support
- mystery hint for upper worlds

## NPC Bond Loop

Stage NPCs should not be disposable quest tools.

```text
stage rule creates encounter
-> temporary cooperation by necessity
-> NPC values and trouble discovered
-> joint fight or shared risk
-> protagonist weighs system optimum against human choice
-> survival, separation, sacrifice, or unresolved loss
-> gallery helps process the result
```

Some losses should not be fixable. The gallery can then act as a counselor rather than a guide.

## Reputation Gap

Inside the transcendent gallery, the protagonist may be a junior. To outside players or local powers, the same protagonist can look like an unknown monster, masked master, or impossible variable.

Use outside perspectives to:

- show how unusual the protagonist looks to non-gallery observers
- prove growth through reactions from people without full context
- create new conflict, not just praise

## Anti-Patterns

- NPCs become emotional only right before sacrifice.
- The gallery jokes through every serious loss.
- Outside viewpoints exist only to compliment the protagonist.
- Gallery support removes the protagonist's need to decide.
