# Retainer Growth Economy

Use this reference when blood, essence, cooldown, slots, or household/territory resources limit retainer growth.

## Resource Types

| Resource | Function |
|---|---|
| blood power | skill use and summon base |
| blood essence | retainer growth and blessing investment |
| summon cooldown | speed limit for new retainer deployment |
| retainer slot or lineage | organization size limit |
| gear and items | role補完 and specialization |
| territory | faction and production base |
| information | remote hunting and risk avoidance |

## Growth Loop

```text
blood power secured
-> retainer summoned
-> unique role confirmed
-> gear, blessing, or skill invested
-> mission assigned
-> result and loot recovered
-> new retainer, sub-retainer, or territory option opens
```

Summoning cost should rise. New retainers are not free power. The protagonist must choose between strengthening current retainers and acquiring new roles.

## Payoff Rules

- Investment should change a retainer's visible role, not only numbers.
- The same resource should create opportunity cost.
- A retainer's weakness or need should be shown before investment.
- Growth should produce a mission where the new function matters.
- Household affection and management judgment can happen in the same action.

## Anti-Patterns

- Every new retainer replaces a previous retainer.
- Resource cost exists only in UI but never changes decisions.
- Growth is a stat screen with no mission payoff.
- The protagonist never remembers retainer preferences, discomfort, or limits.
