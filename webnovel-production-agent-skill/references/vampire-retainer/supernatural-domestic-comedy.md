# Supernatural Domestic Comedy

Use this reference for protagonists whose grand supernatural identity creates inconvenient everyday problems.

## Domestic Vampire Comedy Loop

```text
vampire property or skill discovered
-> protagonist predicts a genre-dangerous result
-> cautious experiment
-> result is not fatal but deeply inconvenient
-> noble dignity distorts outward reaction
-> inner thought and spoken words collide
-> practical workaround or next goal appears
```

The restriction should remain useful beyond one joke. Garlic, sunlight, body size, speech dignity, blood cravings, and sleep cycles should create repeated life friction.

## Inner And Outer Voice Gap

| Inner state | Outer expression |
|---|---|
| worries about living cost and efficiency | noble monarch declaration |
| interprets events as game mechanics | ancient sovereign speech |
| worries about retainers like family | grants blessing as an absolute ruler |
| stays home because it is convenient | appears too lofty to act personally |
| accidental rescue | read as grand design |

Forced aristocratic speech is a narrative constraint. It prevents clean explanation and keeps misunderstanding alive.

## Comedy Rhythm

```text
grand ability or setting
-> modest expectation
-> mundane inconvenient result
-> noble dignity exaggerates response
-> inner monologue lands the joke
```

## Hidden Identity Relationship

Good hidden identity is not only "will they find out?"

Each identity should satisfy a different emotional need.

Example structure:

```text
online comfortable friend
-> real-world S-rank actor fears the progenitor
-> actor owes debt to progenitor
-> protagonist discovers the identities overlap
-> actor does not know friend and progenitor are the same
```

The risk of reveal should be losing the comfortable relationship, not only being caught.
