# Evidence Based Misunderstanding

Use this reference for misunderstanding comedy where the outside world's overestimation is rational from observed evidence.

## Core Principle

Good misunderstanding does not come from everyone being foolish. It comes from partial information, powerful visible results, and prior rumors combining into a reasonable but inflated conclusion.

## Loop

```text
protagonist has simple domestic or practical intent
-> noble speech or retainer framing changes surface meaning
-> retainer executes with overwhelming competence
-> witness observes power and loyalty
-> unseen master is inferred to be stronger
-> rumor spreads through community or institution
-> new faction, enemy, worshipper, or investigator moves
-> protagonist interprets it as a practical next task
```

## Rule Card

```yaml
id: evidence-based-misunderstanding
purpose: Make external overestimation accumulate logically.
procedure:
  - Set the protagonist's real intent as simple and practical.
  - Separate only the facts an outsider can observe.
  - Write the most rational inference from those facts.
  - Let prior rumor or institutional fear inflate the inference by one step.
  - Make the next protagonist action accidentally support the prior inference.
verify:
  - Does the outsider have a reason not to know the truth?
  - Does the misunderstanding match observed evidence?
  - Is the misunderstanding grounded in action and result, not only wording?
```

## External Evidence

Useful observed evidence:

- retainers are overwhelmingly strong
- retainers call the protagonist master, progenitor, monarch, or equivalent
- retainers use different power systems
- incidents are solved without the protagonist appearing
- world-class actors owe the protagonist a debt
- traces of intervention appear at important incidents
- system names, titles, and speech reinforce the false identity

## Anti-Patterns

- Outsiders overestimate the protagonist without evidence.
- Every misunderstanding comes only from a phrase.
- The same institution repeats "the progenitor is amazing" without causing action.
- Truth is hidden only because characters unrealistically avoid basic questions.
