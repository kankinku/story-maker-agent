# Transcendent Gallery Climb Style

```yaml
style_id: transcendent-gallery-mentored-climb
pov:
  primary: first_person
  secondary:
    - npc_observer
    - other_player_observer
core_loop:
  - encounter
  - document
  - ask
  - compare_advice
  - train
  - apply
  - upload_result
  - review
growth:
  primary:
    - skill_understanding
    - combat_judgment
    - conceptual_breakthrough
  secondary:
    - reputation
    - leadership
    - relationships
community:
  role:
    - mentor_network
    - reaction_chorus
    - exposition
    - emotional_support
  rule: advice_must_remain_plural_and_imperfect
```

## General Scene Rhythm

```text
phenomenon observed
-> interpreted through familiar common sense
-> strange point noticed
-> short curse or reaction
-> action decision
```

## Training Scene Rhythm

```text
current technique limit
-> mentor's short instruction
-> repeated execution
-> unnecessary thought removed
-> compressed image or principle
-> technical breakthrough
```

## Gallery Comment Rhythm

```text
normal analysis
-> exaggerated reaction
-> recurring named-user bit
-> expert objection
-> protagonist summary or selection
```

Mix professional and comic comments, but leave a usable conclusion when information matters.

## Checks

- Is gallery advice split into at least two perspectives?
- Did the protagonist choose or modify advice?
- Does video feedback add new information?
- Was preparation accumulated before combat?
- Does the stage NPC have desire beyond a quest function?
- Are gallery-internal evaluation and outside reputation separated?
