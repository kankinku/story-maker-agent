# Dimension Survival Expedition Style

```yaml
style_id: dimension-survival-expedition
pov:
  person: first
  knowledge: limited
  filter: pragmatic-survivor
tone:
  primary: procedural-survival
  secondary:
    - dry_humor
    - post_apocalyptic_melancholy
    - equipment_satisfaction
    - restrained_compassion
narration:
  default_unit:
    - observation
    - interpretation
    - cost_benefit_judgment
    - action
  explanation_method:
    - system_gives_rule
    - expert_gives_mechanism
    - protagonist_translates_to_practical_terms
  emotion_method:
    - reveal_through_rule_breaking_decisions
    - avoid_long_direct_confession
pacing:
  exposition: medium
  action: tactical_and_short
  reflection: brief_unless_relationship_turn
  scene_change_requirement: one_resource_risk_relationship_or_objective_changes
reward:
  preferred:
    - reusable_equipment
    - resource_conversion
    - new_route
    - expert_knowledge
    - companion_capability
  avoid:
    - pure_stat_rewards
    - unbounded_inventory_gain
```

## Dimension Contract

Each dimension must change the protagonist's action verbs, not only scenery.

```yaml
dimension:
  visual_identity: ""
  environmental_hazard: ""
  dominant_enemy_logic: ""
  resource_category: ""
  local_tragedy: ""
  unique_action_verb: ""
```

Examples:

| Dimension Type | Action Verbs | Core Resources |
|---|---|---|
| survival forest | gather, evade, choose | food, water, hide |
| abandoned spaceship | infiltrate, hack, rescue | gear, batteries, expert AI |
| cyberpunk ruin | track, trade, shoot, negotiate | circuits, weapons, specialists |
| biopunk disaster | adapt, escort, harvest materials | bio-tech, treatment materials |

## Tactical Action

Combat is a problem-solving process, not a spectacle.

```text
detect enemy
-> identify type and number
-> analyze terrain and line of fire
-> choose weapon and ammo
-> infer weakness
-> define success condition for first strike
-> execute
-> confirm result
-> secure loot or route
```

The protagonist's mistakes should come from applying familiar survival rules to an unfamiliar dimension.

Examples:

- focuses on a map and misses a floor object
- uses zombie-world instincts and misses cyberpunk CCTV
- understands gear performance but not the local social rule

## Rubric

- Is the current deficit clear?
- Can the reader feel the time limit?
- Is there a reason all resources cannot be taken?
- Is the difference between chosen and abandoned resources meaningful?
- Is previous loot reused?
- Is combat victory the result of preparation and weakness exploitation?
- Does exposition change route or action?
- Does the protagonist, not the expert companion, make the final decision?
- Does trust change through behavior?
- Does the new dimension require a different action verb?
- Does return improve the home-world state?
- Does the reward create the next bottleneck?
