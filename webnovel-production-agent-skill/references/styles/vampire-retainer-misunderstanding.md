## Parallel Loops

### Domestic Vampire Comedy

```text
vampire trait discovered
-> dangerous expectation
-> cautious experiment
-> inconvenient but nonfatal result
-> noble dignity distorts reaction
-> inner and outer voice collide
```

### Retainer Management

```text
blood resource secured
-> retainer summoned or strengthened
-> role confirmed
-> mission assigned
-> result and loot recovered
-> new role, growth, or faction option opens
```

### Misunderstanding Expansion

```text
light intent
-> noble expression
-> retainer interprets as absolute command
-> overwhelming result
-> witness infers unseen master
-> rumor spreads
-> next protagonist action reinforces rumor
```

## Checks

- Is there observed evidence for the misunderstanding?
- Are protagonist intent and external interpretation both logical?
- Are retainers distinct in role, personality, and devotion?
- Did the protagonist make a strategic decision?
- Does a new enhancement have cost or opportunity cost?
- Does outside perspective cause the next event?
- Does the everyday supernatural constraint keep recurring?
