# Resource Choice And Conversion

Use this reference when writing or auditing farming scenes where the protagonist cannot take everything.

## Farming Is Choice

Resource scenes work when abundance collides with limits.

Track:

- remaining time
- distance to exit
- carrying weight and volume
- current gear
- immediate survival utility
- rarity in the home world
- acquisition risk
- processing requirement after return
- maintenance cost

## Decision Formula

```text
practical value
= immediate survival contribution
+ repeat-use potential
+ compatibility with existing gear
+ home-world scarcity
- weight
- acquisition danger
- maintenance cost
```

At least one of these should happen in a farming scene:

- abandon a resource
- swap one resource for another
- delay acquisition
- mark a route for later
- take a damaged or partial version
- spend a resource to secure a better one

## Three Uses Of Good Loot

### 1. Immediate Use

The resource solves a scene-level pressure.

Examples:

- food restores stamina
- hide becomes an improvised bag
- breacher opens a locked door
- tool becomes a temporary weapon

### 2. Home-World Conversion

The resource changes the base or the protagonist's everyday survival.

Examples:

- hide becomes armor
- future scrap combines with old equipment
- machine parts become base infrastructure
- item is disguised for home-world use

### 3. Next Expedition Expansion

The resource changes what can be attempted next.

Examples:

- batteries extend gear uptime
- harness increases carrying capacity
- mobility equipment expands search radius
- technical companion opens locked areas

## Verification

- Is the item used within one or two episodes after acquisition?
- Does it solve a prior problem?
- Does it create a new bottleneck, maintenance need, or dependency?
- Does it combine with another item, place, or person?
- Are unused named items accumulating?
- Did the protagonist make a meaningful choice instead of collecting everything?

## Anti-Patterns

- Inventory list without use scenes.
- Every new item is strictly stronger than prior gear.
- Unlimited carrying capacity removes tension.
- Rewards are pure stats, not tools that alter route, time, risk, or base life.
- The home-world return scene is only accounting.
