# Procedural Exposition

Use this reference when a new dimension introduces unfamiliar technology, monsters, environment rules, or social logic.

## Translate Lore Into Action

```yaml
id: translate-lore-into-action
procedure:
  - The system or expert gives unfamiliar information.
  - The protagonist translates it into current survival terms.
  - The information changes route, equipment, target, risk, or timing.
  - The changed judgment is shown through immediate action.
verify:
  - If the explanation were removed, would the action change?
  - Are system, expert, and narrator repeating the same content?
```

## Channel Roles

### System

Use for objective rule and immediate constraint.

- current environment
- stay time
- lethal hazard
- exit condition
- reward condition

### Expert Or Local Companion

Use for mechanism and options.

- why it is dangerous
- which path is safer
- what equipment can be obtained
- how equipment works
- what the local rule misses

### Protagonist

Use for practical translation.

- "So this means fuel, shelter, and a way through."
- "The problem is not the door. It is the battery after opening it."
- "If I take that, I abandon the water."

## Sentence Rhythm

This mode is not all short sentences.

Recommended rhythm:

```text
observation in a medium sentence
-> mechanism in a longer sentence
-> practical interpretation
-> short conclusion
-> action
```

Long sentences are for:

- gear mechanism
- route and hazard calculation
- world-state interpretation
- resource efficiency comparison

Short sentences are for:

- danger recognition
- unexpected result
- conclusion
- emotional leak
- action start

## Sensory Priority

For a new dimension's first scene, start from survival-linked senses before abstract visual description.

- ruined Earth: rot, dust, dry mouth, stale air
- abundant forest: cold water, sweet pulp, grass smell
- spaceship: metal odor, low vibration, red emergency light
- cyberpunk ruin: chemical smog, neon, electronic noise
- biopunk disaster: acidic air, spores, fluorescent tissue

## Anti-Patterns

- The system explains it, the expert repeats it, and the protagonist summarizes it again.
- Explanation does not change behavior.
- Technical terms remain technical and never become practical choices.
- Worldbuilding appears before the protagonist's route, risk, or resource decision.
